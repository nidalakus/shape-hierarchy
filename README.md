# Shape Hierarchy

A Java OOP project implementing an abstract Shape class with Circle and Square subclasses.

## Classes
- **Shape** - Abstract base class implementing Comparable
- **Circle** - Extends Shape, calculates area and perimeter
- **Square** - Extends Shape, calculates area and perimeter
- **ShapeTester** - Tests all shape operations

## Features
- Inheritance and abstract classes
- Area and perimeter calculations
- Resize method for scaling shapes
- Shape comparison by area

## Language
Java
